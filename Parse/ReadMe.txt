README for Parse
First unzip the Parse file and import it on to eclipse
-In order to use it, please add the text file you want to parse into the project folder.
-Then please change the name of the text file in the arraylist. 
-This is the line to look for: ArrayList <String> Name = new ArrayList<>(Arrays.asList("ResultTJNp.txt", "ResultTXNp.txt"));
-Change "ResultTJNp.txt", "ResultTXNp.txt" to the name of the text file you want.
-After you run it, the it will give you the average time of each file. 
-You can identify the result by the file name in the console.

Note: the parser parse by space so if you have something else as the delimeter, it won't work.
